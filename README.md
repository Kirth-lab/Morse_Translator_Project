# Morse Translator Project

Made by melluche and Kirth-lab, 1st-year students at ENSEA.
The Morse Translator project allows you to translate a Morse coded message by typing your message in the keypads. It also works in the opposite way : you can code your message in Morse language, and the alphabet coded message will show in the screen. 

In the repository, you may find all the PCB files, code and datasheets. We also left the 3D model of our case, where you can put your PCB, and fix the screen on the top of it. Feel free to explore it.

> At this state of the project, we don't know if the PCB works. We will keep you updated.


### Objectives :
- Translate a Morse coded message to an alphabet coded message by typing the touch "." and "-" on the keypads. After coding, the message will be shown on the screen and you will be able to read it.
- Translate an alphabet coded message to a Morse coded message using the keypads. After typing, a buzzer will play the *bips* of the message, and a LED will blink at the same time.
